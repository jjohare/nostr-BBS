QUICK START
================================================================================

1. Install Dependencies:
   npm install
   npx playwright install chromium

2. Start Dev Server:
   npm run dev

3. Run Mobile Tests:
   npm run test:e2e -- tests/e2e/mobile-chat.spec.ts --project="Mobile Chrome"

4. Interactive Mode (Recommended):
   npm run test:e2e -- tests/e2e/mobile-chat.spec.ts --project="Mobile Chrome" --ui

5. View Report:
   npx playwright show-report

WHAT TO TEST
================================================================================

1. Message Composition
   ✓ Tap input field → virtual keyboard appears
   ✓ Type message → text entered correctly
   ✓ Tap send button → message sent, input clears
   ✓ Long press message → context menu (if implemented)

2. Message Display
   ✓ Messages stack vertically (thread view)
   ✓ Images fit viewport width
   ✓ Link previews contained
   ✓ Timestamps readable

3. Channel Switching
   ✓ Channel list displays correctly
   ✓ Back button navigates properly
   ✓ State persists between channels
   
4. Real-time Updates
   ✓ New messages appear automatically
   ✓ Auto-scroll to bottom
   ✓ Typing indicators (if implemented)

5. Performance & UX
   ✓ All buttons ≥ 44x44px (WCAG 2.1)
   ✓ No horizontal scroll
   ✓ Font sizes ≥ 14px
   ✓ Loading states visible

================================================================================
EXPECTED ISSUES TO REPORT
================================================================================

Critical Priority:
- Touch targets < 44x44px (accessibility violation)
- Virtual keyboard hiding input field
- Horizontal scroll on mobile viewport
- Send button not working on tap

High Priority:
- Images overflowing viewport
- Back button not working
- Long press not triggering actions
- Font sizes too small (< 14px)

Medium Priority:
- Emoji picker overflow
- Link preview layout issues
- Typing indicator missing
- Draft not persisting

Low Priority:
- Swipe gestures not implemented
- Timestamp formatting
- Minor spacing issues
